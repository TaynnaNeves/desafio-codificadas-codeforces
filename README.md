## Problema 1 — 71A - Way Too Long Words

### O que o problema pede?

O problema pede para abreviar palavras com mais de 10 caracteres. A abreviação deve conter a primeira letra, a quantidade de letras removidas e a última letra.

### Como eu resolvi?

Utilizei uma estrutura de repetição para percorrer as palavras e uma condição para verificar o tamanho de cada uma. Quando a palavra possuía mais de 10 caracteres, realizei a abreviação solicitada.

### Código

## Problema 2 — 231A - Team

### O que o problema pede?

O problema apresenta três integrantes de uma equipe e informa quais deles acreditam saber resolver cada questão. O objetivo é contar quantas questões possuem pelo menos dois integrantes confiantes na solução.

### Como eu resolvi?

Somei os valores informados para cada questão. Quando a soma era maior ou igual a dois, incrementei um contador.

### Código

Arquivo: 231A.c

## Problema 3 — 282A - Bit++

### O que o problema pede?

O problema simula operações de incremento e decremento em uma variável X e solicita a exibição do valor final.

### Como eu resolvi?

Percorri todas as operações e verifiquei se continham "++" ou "--". Dependendo da operação, incrementei ou decrementei a variável.

Arquivo: 282A.c
